# Ralqlator - Command Line Calculator

## Project Overview

Ralqlator is a feature-rich command-line calculator supporting exact rational arithmetic, standard arithmetic, bitwise operations, multiple number formats, comprehensive mathematical functions, user-defined functions/sequences/constants, and comparison operators.

## Core Modules

### main.rs
- **Responsibility**: Program entry point, CLI parsing, version display
- **Key Functions**:
  - `main()`: Parse arguments, handle version flag, dispatch to calculation or interactive mode
  - `print_version()`: Display version information from git
  - `print_result()`: Format and output results with scientific notation support
  - `print_bitwise_result()`: Format and output bitwise mode results

### cli.rs
- **Responsibility**: Command-line argument and subcommand definitions
- **Arguments**:
  - `expression`: Positional argument for direct calculation
  - `-r, --row`: Expression argument (shorthand)
  - `-x, --hex`: Hexadecimal output
  - `-o, --oct`: Octal output
  - `-b, --bin`: Binary output
  - `-B, --bits`: Bitwise mode
  - `-v, -V, --version`: Display version information
  - `-F, --functions`: Show function help
  - `-O, --operators`: Show operator help
  - `-N, --formats`: Show number format help
  - `-C, --constants`: Show constants help
- **Subcommands**: `functions`, `operators`, `formats`, `constants`, `info`

### calculator.rs
- **Responsibility**: Calculation flow orchestration with AST support
- **Functions**:
  - `calculate()`: Standard mode calculation (now uses AST evaluation)
  - `calculate_with_functions()`: Calculation with user-defined functions/constants
  - `calculate_bitwise()`: Bitwise mode calculation (i64)
  - `calculate_rational()`: Rational-aware calculation [NEW]
  - Legacy shunting_yard pipeline maintained for backward compatibility

### parser.rs [NEW]
- **Responsibility**: AST definition and recursive descent parsing
- **Structures**:
  - `AstNode`: AST node types (Number, FunctionCall, BinaryOp, UnaryOp, etc.)
  - `BinaryOperator`: Arithmetic and bitwise operators
  - `UnaryOperator`: Negation and bitwise NOT
  - `ComparisonOperator`: Comparison operators
- **Key Functions**:
  - `parse_expression()`: Main parsing function
  - `parse_and_eval()`: Parse and evaluate expression
  - `Parser`: Recursive descent parser struct

### value.rs [NEW]
- **Responsibility**: Unified value type system
- **Structures**:
  - `Value`: Enum with Rational, Integer, Float, Boolean variants
- **Key Functions**:
  - Arithmetic operations: `add()`, `sub()`, `mul()`, `div()`, `pow()`
  - Type conversions: `to_integer()`, `to_float()`, `to_rational()`
  - Utility: `is_exact()`, `is_zero()`, `neg()`, `abs()`

### rational.rs [NEW]
- **Responsibility**: Rational number utilities
- **Key Functions**:
  - `parse_decimal()`: Parse decimal string to rational
  - `float_to_rational()`: Convert float to rational approximation
  - `continued_fraction()`: Compute continued fraction expansion
  - `gcd_bigint()`, `lcm_bigint()`: BigInt GCD and LCM

### error.rs [NEW]
- **Responsibility**: Unified error handling
- **Structures**:
  - `CalcError`: Comprehensive error types
  - `ErrorCode`: Error codes for programmatic handling
- **Key Functions**:
  - Convenience constructors: `parse()`, `type_error()`, `domain()`
  - `code()`: Get error code

### token.rs
- **Responsibility**: Lexical analysis
- **Structures**:
  - `Token`: Token types (Number, Identifier, Operator, etc.)
- **Key Functions**:
  - `tokenize()`: Main tokenization function
  - Support for scientific notation, number prefixes, constants

### shunting_yard.rs
- **Responsibility**: Infix to postfix conversion (legacy pipeline)
- **Note**: Still used for backward compatibility, AST parser is preferred

### evaluator.rs
- **Responsibility**: Postfix expression evaluation
- **Functions**:
  - `eval_postfix()`: Standard mode evaluation
  - `eval_postfix_bitwise()`: Bitwise mode evaluation
  - `format_comparison_result()`: Format comparison results

### functions.rs
- **Responsibility**: Mathematical function implementations
- **Categories**:
  - Logarithms, roots, powers
  - Trigonometric and hyperbolic functions
  - Special functions (gamma, erf, erfc, beta)
  - BigInt functions (bfactorial, bpow, comb, perm, gcd, lcm, isprime, nextprime)
  - Rational functions (num, den, frac, rational, float) [NEW]

### repl.rs
- **Responsibility**: Interactive REPL
- **Features**:
  - History navigation, Tab completion
  - Last result insertion (`@`)
  - Mode switching
  - Comprehensive help system
  - User-defined functions/sequences/constants
